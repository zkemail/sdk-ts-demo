# sdk-ts-demo

Demo of using the typescript zk-email sdk

## Run a node_js demo script

This also works using `bun`.

```bash
cd node_js
bun i
bun src/shortest-example.ts
```

## Run NextJS demo

```bash
cd nextjs
pnpm i
pnpm run dev
```

## Run vite vanilla js demo

```bash
cd vite
pnpm i
pnpm run dev
```
